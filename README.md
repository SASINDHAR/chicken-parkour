# Chicken Parkour: Escape the KFC

A tiny, hilarious — and brutally hard — 2D side-scroller made with HTML5 Canvas. Controls: **Space** or **Up** to jump.

## Play
Open `index.html` in your browser or enable GitHub Pages (see below).

## How to run locally
```bash
# if you want a simple local server (recommended)
python -m http.server 8000
# then open http://localhost:8000
